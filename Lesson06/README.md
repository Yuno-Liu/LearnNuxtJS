## 实战演习

### static/robots.txt

```
User-agent: *
Disallow: /article/
Allow: /

Sitemap: https://xxx.xxx.xxx/sitemap.xml
```

### static/landing_page.html

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>着陆页面</title>
</head>
<body>
    <h1>这里是着陆页面。(广告效果)</h1>
</body>
</html>
```
